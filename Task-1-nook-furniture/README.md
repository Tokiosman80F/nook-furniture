# Nook Furniture Website

A minimalistic and futuristic furniture website built to match the Figma design exactly.

## Setup Instructions

### Images
The images have been downloaded from Figma and should be placed in the `assets/images/` directory. If the images are not present, they were downloaded to `C:\Users\User\assets\images\` and need to be copied to this project's `assets/images/` folder.

Required images:
- hero-image.svg
- header-logo.svg
- arrow-icon.svg
- arrow-icon-2.svg
- decorative-shape.svg
- arrow-up-right.svg
- furniture-stats.svg
- decorative-shape-2.svg
- product-1.svg through product-8.svg
- support-icon.svg
- shipping-icon.svg
- support-icon-2.svg
- social-icons.svg
- logo.svg
- chevron-left.svg
- chevron-right.svg

### Fonts
The website uses the following fonts:
- **Roslindale** - For headings (serif font)
- **Inter** - For body text (sans-serif)
- **Wilkysta** - For decorative text

Note: Roslindale and Wilkysta may need to be installed locally or loaded from a font service. The CSS includes fallback fonts.

### Running the Website
Simply open `index.html` in a web browser. No build process is required.

## Design Specifications

### Colors
- Primary: #00494A
- Primary Dark: #02494B
- Background: #FAFAFA
- White: #FFFFFF
- Text Primary: #303635
- Text Secondary: #5F6060

### Layout
- Max Width: 1440px
- Responsive design included for mobile devices

## Sections

1. **Hero Section** - Main landing area with headline and product cards
2. **Navigation Bar** - Category navigation
3. **Category Cards** - Visual category showcase
4. **Futura Section** - Feature section with statistics
5. **Popular Collection** - Product grid with filtering
6. **Why Us Section** - Feature highlights
7. **Contact Section** - Contact form
8. **Footer** - Links and company information
